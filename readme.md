## Calculator Web Application
```
Một simple calculator web application built with HTML, CSS, Javascript
```
## Overview 
```
This project is a simple calulator with basic operation including : 
- Addition (+)
- Substraction (-)
- Multiplication (*)
- Division (/)
- Delete (DEL)
- All clear (AC)
- Ans, PreAns
- Percentage (%)
- Log10 (log)
- Square (x^2)
- Pow (x^y)
- Sqrt 
- Factorial (!)
```

## Technology Used 
```
- HTML5
- CSS3
- Javascript
```
